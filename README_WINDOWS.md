# Antigravity Causal — Windows Guide 🚀

This guide will help you install and run **Antigravity Causal** on your Windows computer in the most user-friendly way possible.

---

## 1. Prerequisites (First Time Only)

Make sure you have **Python** installed on your computer.

1.  Go to [python.org/downloads](https://www.python.org/downloads/windows/)
2.  Click the big yellow **"Download Python 3.x.x"** button.
3.  **IMPORTANT:** When the installer opens, **check the box** that says **"Add Python to PATH"**.
4.  Click **"Install Now"**.

---

## 2. How to Start the App 🚦

Starting the app is as simple as **double-clicking a file**.

1.  Open the folder containing this project.
2.  Find the file named **`launcher.bat`**.
3.  **Double-click it.**

### What will happen?
- A black window (terminal) will open.
- The script will automatically set up everything (first time takes a minute).
- **Your Web Browser will open automatically** to `http://127.0.0.1:8000`.
- You can now use the app!

---

## 3. Create a Desktop Shortcut ⭐️

For even easier access, you can put the app right on your desktop:

1.  **Right-click** on the `launcher.bat` file.
2.  Select **Send to** → **Desktop (create shortcut)**.
3.  Go to your desktop, **right-click** the new shortcut, and select **Rename**.
4.  Type **"Antigravity Causal"**.
5.  (Optional) **Right-click** → **Properties** → **Change Icon** to make it look pretty!

---

## 4. Troubleshooting 🛠️

**The black window closes immediately?**
- Make sure you checked the "Add Python to PATH" box during Python installation.
- Try running `launcher.bat` again.

**The webpage doesn't open?**
- Wait about 10 seconds.
- If it still doesn't open, type `http://127.0.0.1:8000` into your browser's address bar.

**I see "Error: Could not find requirements.txt"**
- Make sure you haven't moved `launcher.bat` out of its folder. It needs to stay in the same folder as the other files.

---

Enjoy using the app! If you have any questions, feel free to ask.
